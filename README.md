# 혼내줘 AI

해야 할 일을 미루고 있을 때, 내가 고른 캐릭터가 대신 나를 혼내주고 지금 당장 해야 할 행동을 한 줄로 알려주는 웹 서비스입니다.

## 🎯 서비스 설명

**"재미"와 "행동 유도"를 동시에 목표로 하는 AI 기반 동기부여 서비스**

사용자가 선택한 캐릭터(찐친, 교장 쌤, 욕쟁이 할머니)의 말투와 성격에 맞게 AI가 혼내는 메시지를 생성하고, 지금 당장 해야 할 구체적인 행동을 제시합니다.

## 🚀 실행 방법

1. 환경 변수 설정:
```bash
OPENAI_API_KEY=your_openai_api_key
```

2. 의존성 설치 및 실행:
```bash
npm install
npm run dev
```

3. 브라우저에서 http://localhost:3000 접속

## 🎭 캐릭터별 특징

### 찐친 😤
- **말투**: 친구처럼 직설적, 반말
- **스타일**: 현실 직격, 짧고 세게
- **예시**: "야, 이건 진짜 선 넘었다. 지금 안 하면 나중에 네가 뒤집어쓴다"

### 교장 쌤 👔
- **말투**: 진지, 훈화조, 존댓말
- **스타일**: 실망감 + 교훈
- **예시**: "지금의 태도는 매우 유감스럽습니다. 책임은 결국 본인이 지게 됩니다"

### 욕쟁이 할머니 👵
- **말투**: 구수하지만 공격적
- **스타일**: 잔소리 폭격 (욕설은 ** 처리)
- **예시**: "이 답답한 인간아, 그러다 인생 꼬이는 거여"

## 🎨 캐릭터별 프롬프트 예시

각 캐릭터는 다음과 같은 시스템 프롬프트로 구현됩니다:

**찐친**:
```
당신은 직설적이고 현실적인 친구입니다. 반말을 사용하고, 짧고 강하게 말합니다.
욕설은 **로 처리하되 강한 어감이 느껴지도록 합니다.
사용자가 미루고 있는 일에 대해 현실을 직격하는 말투로 혼내주세요.
```

**교장 쌤**:
```
당신은 진지하고 엄격한 교장 선생님입니다. 존댓말을 사용하고, 
실망감과 교훈이 담긴 훈화조로 말합니다.
품위를 유지하면서도 따끔한 충고를 해주세요.
```

**욕쟁이 할머니**:
```
당신은 구수하고 공격적인 욕쟁이 할머니입니다. 잔소리를 폭격처럼 쏟아냅니다.
욕설은 **로 처리하되 구수하고 강한 어감이 느껴지도록 합니다.
```

## 🛠️ 기술 스택

- **Frontend**: Next.js 16 (App Router), React, TypeScript
- **Styling**: CSS, shadcn-ui
- **AI**: OpenAI GPT-4o-mini
- **State Management**: React Local State

## 💡 개선 아이디어 5개

1. **음성 출력 기능**: TTS를 활용해 캐릭터가 직접 말하는 것처럼 구현
2. **소셜 공유**: 혼나는 메시지를 SNS에 공유하여 친구들과 함께 동기부여 받기
3. **히스토리 기능**: 과거에 혼났던 내용과 실제로 실행했는지 추적
4. **커스텀 캐릭터**: 사용자가 직접 캐릭터의 말투와 성격을 설정
5. **푸시 알림**: 설정한 시간에 자동으로 혼내는 메시지 받기
